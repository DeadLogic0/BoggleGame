GAMEPLAY
    Entering Words: Click on tiles to select them. Only adjacent tiles can be selected after the first selection.
                    To check if a string of tiles is a word click on the last tile again after selecting it.
    Timer: The green bar at the bottom of the screen represents the remaining time. The smaller it is the less
           time is remaining.
    Guessed Words: Guessed words will appear to the right of the board. Red if incorrect and Green if Correct.
    New Game: Click New 5x5 for a 5x5 Game and New 4x4 for a 4x4 Game.
Bugs
